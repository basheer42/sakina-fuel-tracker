What is TR830?

TR830 is a KRA document that shows captured/received entries per product per destination. It contains vessel information, product types (PMS/AGO), quantities, and destination details. Our enhanced parser can extract this data automatically and create shipment records.

  • • Supports PDF format only
  • • Maximum file size: 10MB
  • • Automatically detects product types (PMS/AGO)
  • • Creates shipments with FIFO stock management
